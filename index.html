⚙️ Configuración

Cómo obtener la URL del script:
1. Abrí script.google.com
2. Pegá el archivo BACKEND_GoogleAppsScript.js
3. Reemplazá REEMPLAZA_CON_TU_ID_DE_CARPETA con el ID de tu carpeta de Drive
4. Implementar → Nueva implementación → Aplicación web
5. Ejecutar como: Yo · Acceso: Cualquier usuario
6. Copiá la URL y pegala abajo
